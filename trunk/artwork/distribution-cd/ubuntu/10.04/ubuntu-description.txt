* Ubuntu Desktop คืออะไร?

Ubuntu เป็นระบบปฏิบัติการที่สามารถใช้แทน Windows ได้อย่างสมบูรณ์แบบ มันรวมเอาไว้ด้วยโปรแกรมโอเพนซอร์สกว่าพันโปรแกรม และสามารถทำงานได้ทั้งที่บ้านและกับงานสำนักงาน

Ubuntu ไม่มีปัญหาเรื่องไวรัสเหมือนอย่าง Windows คุณสามารถท่องอินเทอร์เน็ตได้อย่างปลอดภัยด้วย Firefox สร้างเอกสารและตารางคำนวน เล่นเกม ดูหนัง ฟังเพลง และอีกหลายอย่างที่คุณต้องการทำ มันมีพร้อมด้วยคลังเพลงและบริการจัดเก็บและแบ่งปันเอกสาร และยังช่วยให้คุณติดต่อกับเพื่อนบน Facebook และ Twitter ได้อย่างง่ายดายผ่านหน้าจอของคุณ

ด้วยซีดีแผ่นนี้ คุณสามารถติดตั้ง Ubuntu ได้ พร้อมด้วยการรองรับภาษามากมายในโลก และด้วยการสนับสนุนระยะยาวของรุ่นนี้ (Long Term Support) ที่จะให้การสนับสนุนถึง 3ปี ในระบบการใช้งานทั่วไป และ 5ปี ในระดับเซิร์ฟเวอร์

Ubuntu พัฒนาโดยชุมชนผู้ใช้ทั่วโลก อย่าลืมว่าคุณสามารถดาวน์โหลดรุ่นที่ออกแบบมาเพื่อคอมพิวเตอร์เน็ตบุคโดยเฉพาะได้ที่ www.ubuntu.com

* ท่องดูและติดตั้ง
